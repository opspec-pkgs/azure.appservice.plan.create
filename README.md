# Problem statement

creates an azure app service plan (if it doesn't already exist)

# Example usage

## Visualize

```shell
opctl ui github.com/opspec-pkgs/azure.appservice.plan.create#2.1.0
```

## Run

```
opctl run github.com/opspec-pkgs/azure.appservice.plan.create#2.1.0
```
